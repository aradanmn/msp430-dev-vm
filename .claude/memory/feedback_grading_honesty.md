When grading exercises, do not give credit for code that was provided as boilerplate/template. Scott will call this out — he distinguishes between what he wrote and what was given to him.

**Why:** Scott flagged that the clean delay_ms in Ex3 was provided, not his work. He values honest assessment over inflated grades.

**How to apply:** Before praising code in a grade report, verify it was actually written by the student, not part of the exercise template. Check the exercise spec for what was "provided" vs what the student was asked to write.
